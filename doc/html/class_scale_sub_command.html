Detailed Description

Hidden scale subcommand to call from console.

Author
Jakub Baldyga
Date
16/12/2023
Copyright
(c) 2023 Jakub Baldyga, MIT License

Description:
Provides functionality to call the scale method from console, which scales the given image
Call with –help for more details and available commands

MathDraw hidden scale --help

Usage:

MathDraw hidden scale -i <path to image> -w <width> -h <height> -o <output file>
